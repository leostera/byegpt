ByeGPT Privacy Policy

Effective date: March 24, 2026

ByeGPT is a Chrome extension that exports the ChatGPT conversations already visible in the current signed-in browser session into local files. This policy describes what the extension processes and how that data is handled.

What the extension processes

The extension may process:

  • conversation content visible to the current signed-in ChatGPT user
  • conversation titles and conversation identifiers
  • JSON API responses used by the ChatGPT web application for those conversations
  • asset URLs and downloaded asset files referenced by those conversations
  • local crawl state, such as which conversations have been discovered or downloaded

How the data is used

The extension uses this data only to:

  • build an inventory of the current user's visible conversations
  • export those conversations into local JSON files
  • download conversation-related assets into local folders
  • show progress information while the export is running

Where the data goes

ByeGPT is designed to work locally in the browser. Exported conversation data is stored in Chrome extension storage during processing and then written to the user's local Downloads folder. The project does not operate a remote server that receives exported conversation data.

Sharing and sale of data

This project does not sell exported data. It does not intentionally share exported conversation content with third parties, except for the normal browser and operating system behavior needed to save files to the user's local machine.

Retention

Local extension state remains in Chrome storage until the user clears it, restarts the extension crawl state, or uninstalls the extension. Files written to the Downloads folder remain there until the user deletes them.

Security

ByeGPT uses the browser's local extension storage and download mechanisms. No method of storage or transmission is perfect, so users should review exported files and handle them according to their own security requirements.
