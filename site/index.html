Chrome Extension

ByeGPT

ByeGPT exports the ChatGPT conversations already visible in your signed-in browser session into local JSON files, along with referenced assets.

Local-first export

Captured conversation data is stored in the browser and written to local files. This project does not operate a remote backend for exported chat data.

Session scoped

The extension only exports the chats your current signed-in ChatGPT user can already access in the browser.

Structured output

Each conversation is exported as its own JSON file, with related assets written under a matching conversation asset folder.
